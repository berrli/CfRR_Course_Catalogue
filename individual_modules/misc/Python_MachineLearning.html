Introduction to Machine Learning#

Definition#

Machine Learning (ML) is a subfield of Artifical Intelligence (AI), a branch of computer science that looks to create system capable of performing tasks that normally require typically human intelligence [MMRS06]. ML aims to enable computer to perform a range of tasks without explicit instruction, formally defined as “A computer program is said to learn from experience E concerning some class of tasks T and performance measure P if its performance at tasks in T, as measured by P, improved with experience E.” [Lea97].

Given the following definition an example might be the task of classifying incoming emails as either “spam or “not spam”, in the previous definition this would be the task T. In this case the experience, E, would be the dataset of all the previous emails that have been recieved, with the assoicated classification of either “spam” or “not spam”. The goal with the experience would be to identify patterns in the “spam” or “not spam” email, pattern that could be used include specific keywords, senders email addresses, the structure of the email and other metadata that helps you group the data together. The performance measure P, would in this case be accuracy, which is the proportion of email correctly classified by the model.

Types of Machine Learning#

Supervised Learning#

Supervised machine learning is where the algorithm is trained on a label dataset, such as the example above with the labelled “spam” or “not spam” emails. In this case each input data point in the dataset is paired with an output label, which gives the correct answer for that data points. The goal therefore of supervised machine learning is to learna mapping from the input, the email and its assoicated features to outputs. The overall goal is to then be able to provide the input for a data point it has never seen before and map it to the correct output. In the above example, providing the model with a new email it has never seen before and for it to correctly label it as either “spam” or “not spam”.

Unsupervised Learning#

Unsupervised machine learning is where the algorithm is trained on a dataset without predefined labels or explicit instrcutions on what to look for in the data. The main objctives of unsupervised learning is to idnentify underlying patterns, structure or relationships in the data that can help in drawing conclusion or making decisions. In the above example it could be the case that clustering, a machine learning technique that we will discuss later is used in the email example. If two clusters were desired, the algorithm would try to partiion the data into two distinct groups that could be “spam” or “not spam”. However it is possible that more than two clusters are used, and it partition the data based on other types of features, such as the style of some spam emails being similar and so they are grouped together, having found underlying paterns in their writing style

There are also some other types of machine learning, such as reinforcement learning, semi-supervised learning etc, but in the context of this course we will focus on the simple paradigm of algorithm being either supervised or unsupervised.

Real-world Applications of Machine Learning#

Machine learning is being used incredibly widely in society today, some example include:

  1. Healthcare

    • Disease Diagnosis: ML models are used to diagnose diseases more accurately and quickly than traditional methods. For example, deep learning models analyze medical images for signs of diseases such as cancer, diabetic retinopathy, or heart diseases [ALS22].

    • Drug Discovery: ML algorithms help in predicting molecule interactions, thus speeding up the drug discovery process and lowering development costs [VCC+19].

    • Personalized Medicine: Machine learning techniques can analyze data from patient histories to tailor medical treatments to individual patients [PJDonnesC21].

  2. Finance

    • Fraud Detection: Machine learning models are employed to detect unusual patterns of behavior that suggest fraudulent activities on accounts [AAO17].

    • Algorithmic Trading: ML algorithms are used to make high-speed trading decisions based on patterns detected in market data [Jan20].

    • Credit Scoring: ML models are increasingly used to assess a person’s creditworthiness by analyzing traditional and alternative data sources [DCP20].

  3. Retail

    • Recommendation Systems: ML powers personalized recommendations on platforms like Amazon and Netflix, improving customer experience and increasing sales [Mic17].

    • Inventory Management: Predictive analytics are used for better forecasting of inventory needs, optimizing stock levels based on predicted demand [DFM23].

    • Customer Sentiment Analysis: Analyzing customer feedback and reviews to derive insights about customer satisfaction and preferences [YL20].

  4. Automotive and Transportation

    • Self-driving Cars: Machine learning models process data from vehicle sensors to make decisions in real time, enabling autonomous driving [RF18].

    • Route Planning: Algorithms analyze traffic data to suggest optimal routes, reducing travel time and avoiding congested areas [SMerchanW20].

    • Predictive Maintenance: Using data from vehicle sensors to predict when a vehicle or part is likely to fail [CSV+19].

  5. Telecommunications

    • Network Optimization: ML algorithms optimize network quality and performance by predicting and preventing network failures [WCW+17].

    • Customer Churn Prediction: Machine learning helps in predicting which customers are likely to leave, allowing companies to take proactive actions to retain them [FSK+22].

  6. Energy

    • Smart Grid Management: Machine learning is used to enhance the efficiency of power generation and distribution, and to predict equipment failures [ASW19].

    • Energy Consumption Optimization: ML algorithms optimize energy usage in real-time by analyzing data from smart meters [PPC+20].

  7. Education

    • Adaptive Learning Technologies: Machine learning tailors educational experiences to individual student’s learning pace and style, enhancing engagement and outcomes [ECHH23].

    • Predicting Student Performance: Algorithms can predict student dropouts and academic success, which helps in intervening early to offer support [XMVDS17].

  8. Agriculture

    • Crop and Soil Monitoring: ML models help in monitoring crop health and soil conditions, predicting agricultural outputs which can aid in sustainable farming practices [MSS+20].

    • Precision Farming: Machine learning is used to analyze data from farm sensors and satellite images to make farming more precise and productive [SMRS22].
